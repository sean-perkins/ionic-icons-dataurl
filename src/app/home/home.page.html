The following ion-icon has an embeeded imagen in the src. You can't see it now, but this used to work with previous versions of ionic/Angular. The src is now truncated.

Update: after installing many versions. I've found that the only version that works fine is v6.0.0. Starting from v6.0.1 and up to the latest v6.5.0 the ion-icon src value gets truncated to 30 characters. I made my experiments locally and replicated this in stackblitz too:

In conclusion, with Angular v14:
  • "@ionic/angular": "6.0.0", -> works
  • "@ionic/angular": ">=6.0.1", -> fails
